## 二、问题详情

### 🔴 高优先级 (需立即修复)

#### 1. 安全密钥硬编码
**位置**: `backend/app/core/config.py`

```python
SECRET_KEY: str = "your-secret-key-change-in-production"
```

**风险**: 生产环境使用默认密钥，容易被攻击者利用

**建议**: 
- 使用环境变量或配置文件
- 生成随机密钥

**修复方案**:
```python
import os
SECRET_KEY: str = os.getenv("SECRET_KEY", "fallback-key-change-in-production")
```

#### 2. 密码哈希算法不安全
**位置**: `backend/app/core/security.py`

```python
def verify_password(plain_password: str, hashed_password: str) -> bool:
    salt, stored_hash = hashed_password.split("$", 1)
    computed_hash = hashlib.sha256((salt + plain_password).encode()).hexdigest()
```

**风险**: SHA256不安全，应使用bcrypt或argon2

**建议**: 使用 `passlib` + `bcrypt`
